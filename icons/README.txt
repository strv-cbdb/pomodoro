icons/ ディレクトリ

このディレクトリには PWA 用のアイコン画像を配置してください。

必要なファイル:
  icon-192.png  ... 192x192 px のアプリアイコン
  icon-512.png  ... 512x512 px のアプリアイコン

要件:
  - 形式: PNG
  - 背景: 黒 (#000000) 推奨 (アプリのテーマカラーに合わせる)
  - 内容: アプリのロゴやシンボルを配置してください
  - purpose: "any maskable" として登録済み (manifest.json 参照)
            maskable の場合、画像の周囲 10% は安全マージン外になります

現在の状態:
  このプレースホルダーファイルのみ存在します。
  実際の PNG ファイルを作成・配置してから GitHub Pages に公開してください。

アイコン未配置の場合:
  PWA のインストールは可能ですが、ホーム画面アイコンが表示されません。
  開発中はこのままでも動作します。
